Определены следующие управляющие последовательности:
~~              Ввести один знак ~
~c адресаты     Добавить адресатов в список рассылки копий
~d              Вставить файл dead.let
~e              Редактировать набранное сообщене
~h              Редактировать заголовок сообщения
~r файл         Читать файл в набираемое сообщение
~p              Вывести набранное сообщение
~m сообщения    Вставить сообщения, сдвинув их на табуляцию
~s тема         Установить поле темы в заголовке
~t адресаты     Добавить адресатов в список рассылки
~v              Редактировать сообщение экранным редактором
~u [исх] пол    Закодировать файл UUENCODE
~w файл         Записать набранное сообщение в файл
~?              Печатает этот текст
~>		Читать сообщение в набираемое, вставляя >
~!команда       Выполнить команду системы
~|команда       Прогнать сообщение через команду (фильтр)
------------------------------------------------------------
